General info



API Access


In order to use this API, a valid access token is needed, request one here,
it will be send to your mail within a few minutes.

Pagination


The API returns paginated results, with a default limit set to 100 results. To be able to paginate to next or previous pages, use the previous and next links provided in the JSON Response. Below is an example JSON response, containing pagination details.

Sorting


Sorting on results is possible by providing the sort key, also order can be applied.

Here is an example thats sorts Events by date:

/api/v1/events?sort=date&order=desc&token=KEY

Field selection


Selection on which fields should be shown on the response result is possible by providing the fields key.

Here is an example query thats shows Events with only a title and date:

/api/v1/events?fields=title,date&token=KEY
