Flow API - Endpoints

Album

Path Method Parameters Description
api/v1/albums GET None Retrieve all albums details.
api/v1/albums/albumId GET String, albumId Retrieve album details for a specific album.
api/v1/albums/artist/artistId GET String, artistId Retrieve album details for a specific artist.

Artist

Event

Path Method Parameters Description
/api/v1/events GET None Retrieve all events details
/api/v1/events/eventId GET String, eventId Retrieve event details for a specific event
/api/v1/events/date/eventDate GET String, eventDate Retrieve event details for a specific event on a specific date
/api/v1/events/genre/genreId GET String, genreId Retrieve events details based on a specific genre
/api/v1/events/artist/artistId GET String, artistId Retrieve events details for a events where a specific artist is performing.
