Album

Path Method Parameters Description
api/v1/albums
 GET None Retrieve all albums details.
api/v1/albums/albumId
 GET String, albumId Retrieve album details for a specific album.
* api/v1/albums/artist/artistId
 GET String, artistId Retrieve album details for a specific artist.
